Welcome to the Simple Wikipedia Clone

This is a simple clone of Wikipedia built using only HTML and CSS. It includes a basic layout with a header, sidebar for navigation, and a main content area for articles.

Feel free to explore the different sections by clicking the links in the sidebar. Each link could potentially take you to a different article, but for now, this is just a static page example.

Wikipedia is a free online encyclopedia that was launched in 2001. It allows users to contribute articles on a wide range of topics, making it one of the largest sources of knowledge on the web.
